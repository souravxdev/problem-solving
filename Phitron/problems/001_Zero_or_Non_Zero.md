# Zero or Non Zero

**Time Limit:** 1 second  
**Memory Limit:** 2.93 MB  

### Problem Statement
You are given an integer number `N`.  
Print **"Zero"** if the number equals `0`, and **"Non Zero"** otherwise.

### Constraints
`-1000 ≤ N ≤ 1000`

### Input Format
- The input consists of a single integer `N`.

### Output Format
- Print `Zero` if the number is equal to `0`.  
- Print `Non Zero` otherwise.

### Sample Input 1
`5`

### Sample Output 1
`Non Zero`

### Explanation
The input value is `5`, which is not zero.  
Therefore, the output should be **Non Zero**.
