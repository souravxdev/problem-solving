# Divisible By 5 or Not

**Time Limit:** 1 second  
**Memory Limit:** 2.93 MB  

### Problem Statement
You will be given a positive integer `N`.  
You need to print numbers from `1` to `N`, and beside each number, print **"Yes"** if it is divisible by 5, or **"No"** otherwise.  

### Constraints
`1 ≤ N ≤ 1000`

### Input Format
- The input will contain a single positive integer `N`.

### Output Format
- For each number from `1` to `N`, print the number followed by **"Yes"** or **"No"** as specified.  
- Put a new line after every line.

### Sample Input 1
`10`

### Sample Output 1
    1 No
    2 No
    3 No
    4 No
    5 Yes
    6 No
    7 No
    8 No
    9 No
    10 Yes

### Explanation
Each number from `1` to `10` is printed.  
Only those divisible by 5 are followed by **"Yes"**, while others are followed by **"No"**.
