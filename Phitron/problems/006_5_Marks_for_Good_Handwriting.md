# 5 Marks for Good Hand Writing

**Time Limit:** 1 second  
**Memory Limit:** 2.93 MB  

### Problem Statement
In this problem, you will be given an integer number `N`.  
You need to add **5** to `N` and print the result.

### Constraints
`1 ≤ N ≤ 100`

### Input Format
- The input consists of a single integer `N`.

### Output Format
- Output the result after adding **5** to `N`.

### Sample Input 1
`100`

### Sample Output 1
`105`

### Explanation
The input is `100`.  
After adding `5` to `100`, the result becomes `105`.
