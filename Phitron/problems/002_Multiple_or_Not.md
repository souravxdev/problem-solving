# Multiple or not

**Time Limit:** 1 second  
**Memory Limit:** 2.93 MB  

### Problem Statement
In this problem, you will be given two integers `a` and `b`.  
You have to determine whether one number is a multiple of the other — that is, check if **`a` is a multiple of `b`** or **`b` is a multiple of `a`**.

### Constraints
`1 ≤ a, b ≤ 10000`

### Input Format
- The input will contain two integers `a` and `b`.

### Output Format
- Print **"Yes"** if one of the numbers is a multiple of the other.  
- Print **"No"** otherwise.

### Sample Input 1
`5 9`

### Sample Output 1
`No`

### Explanation
Neither 5 is a multiple of 9, nor 9 is a multiple of 5.  
Hence, the output is **No**.
