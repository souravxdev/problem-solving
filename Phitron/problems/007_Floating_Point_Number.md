# Floating Point Number

**Time Limit:** 1 second  
**Memory Limit:** 2.93 MB  

### Problem Statement
In this problem, you will be given a floating-point number `x`.  
You have to print the number `x` up to **3 decimal places**.

### Constraints
`1.0 ≤ x ≤ 100.0`

### Input Format
- The input will contain a single floating-point number `x`.

### Output Format
- Print the number `x` up to **3 decimal points**.

### Sample Input 1
`22.12344`

### Sample Output 1
`22.123`

### Explanation
The given number `22.12344` is printed up to **3 decimal places**, resulting in `22.123`.
