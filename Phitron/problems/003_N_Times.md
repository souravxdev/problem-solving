# N Times

**Time Limit:** 1 second  
**Memory Limit:** 2.93 MB  

### Problem Statement
You will be given a positive integer `N`.  
You need to print **"I Love Practice"** exactly `N` times.
Here, a *positive integer* means an integer greater than `0`.

### Constraints
`1 ≤ N ≤ 1000`

### Input Format
- You will be given a single positive integer `N`.

### Output Format
- Output **"I Love Practice"** `N` times.  
- Don’t forget to print each occurrence on a new line.

### Sample Input 1
`5`

### Sample Output 1
    I Love Practice
    I Love Practice
    I Love Practice
    I Love Practice
    I Love Practice


### Sample Input 2
`2`

### Sample Output 2
    I Love Practice
    I Love Practice


### Explanation
If `N = 5`, the phrase **"I Love Practice"** is printed 5 times, each on a new line.
