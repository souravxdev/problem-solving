# Max and Min
**Sheet #1 - Data Type - Conditions**

**Time Limit:** 0.25 second  
**Memory Limit:** 64 MB  

### Problem Statement
Given three numbers `A`, `B`, and `C`, print the **minimum** and the **maximum** among them.

### Constraints
`-10^5 ≤ A, B, C ≤ 10^5`

### Input Format
- Only one line containing three integers `A`, `B` and `C`.

### Output Format
- Print the minimum number, followed by a single space, then print the maximum number.

### Examples
### Input 1
`1 2 3`

### Output 1
`1 3`

### Input 2
`-1 -2 -3`

### Output 2
`-3 -1`

### Input 3
`10 20 -5`

### Output 3
`-5 20`

### Explanation
- In the first example, the smallest value is `1` and the largest value is `3`.  
- In the second example, among `-1, -2, -3`, the smallest is `-3` and the largest is `-1`.  
- In the third example, the minimum is `-5` and the maximum is `20`.
