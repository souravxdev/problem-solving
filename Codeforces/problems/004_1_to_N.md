# 1 to N
**Sheet #2 - Loops**

**Time Limit:** 1 second  
**Memory Limit:** 256 MB  

### Problem Statement
Given a number `N`, print all numbers from **1 to N**, each on a new line.

### Constraints
`1 ≤ N ≤ 10^3`

### Input Format
- Only one line containing the integer `N`.

### Output Format
- Print `N` lines where each line contains one integer from **1 to N**.

### Examples
### Input 1
`5`

### Output 1
`1
2
3
4
5`

### Explanation
The task is simple: print numbers starting from 1 up to the given number `N`, each on its own line.
