# Welcome for You with Conditions
**Sheet #1 - Data Type - Conditions**

**Time Limit:** 1 second  
**Memory Limit:** 64 MB  

### Problem Statement
You are given two numbers `A` and `B`.  
Print **"Yes"** if `A` is greater than or equal to `B`. Otherwise, print **"No"**.

### Constraints
`0 ≤ A, B ≤ 100`

### Input Format
- Only one line containing two numbers `A` and `B`.  

### Output Format
- Print **"Yes"** if `A ≥ B`. 
- Otherwise, print **"No"**.

### Examples
### Input 1
`10 9`

### Output 1
`Yes`

### Input 2
`5 5`

### Output 2
`Yes`

### Input 3
`5 7`

### Output 3
`No`

### Explanation
- In the first example, `10 ≥ 9`, so the output is **Yes**.  
- In the second example, both numbers are equal (`5 = 5`), so the output is also **Yes**.  
- In the third example, `5` is less than `7`, so the output is **No**.
