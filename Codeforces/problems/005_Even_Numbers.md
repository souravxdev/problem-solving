# Even Numbers
**Sheet #2 - Loops**

**Time Limit:** 1 second  
**Memory Limit:** 256 MB  

### Problem Statement
Given a number `N`, print all **even numbers** between `1` and `N` (inclusive), each on a new line.

### Constraints
`1 ≤ N ≤ 10^3`

### Input Format
- Only one line containing the integer `N`.

### Output Format
- Print all even numbers between `1` and `N` inclusive.  
- If there are **no even numbers**, print `-1`.

### Examples
### Input 1
`10`

### Output 1
`2
4
6
8
10`

### Input 2
`5`

### Output 2
`2
4`

### Explanation
- For input `10`, the even numbers between `1` and `10` are printed.  
- For input `5`, the even numbers between `1` and `5` are `2` and `4`.  
- If `N = 1` (no even numbers), the output should be `-1`.
