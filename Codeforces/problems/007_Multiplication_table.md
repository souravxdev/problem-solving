# Multiplication Table
**Sheet #2 - Loops**

**Time Limit:** 1 second  
**Memory Limit:** 64 MB  

### Problem Statement
Given a number `N`, print the **multiplication table** of `N` from `1` to `12`.  
Each result must be printed in the format:

`N * i = result`

### Constraints
`1 ≤ N ≤ 50`

### Input Format
- Only one line containing the integer `N`.

### Output Format
- Print **12 lines**, each showing the multiplication of `N` with numbers from `1` to `12`.

### Examples
### Input 1
`1`

### Output 1
`1 * 1 = 1
1 * 2 = 2
1 * 3 = 3
1 * 4 = 4
1 * 5 = 5
1 * 6 = 6
1 * 7 = 7
1 * 8 = 8
1 * 9 = 9
1 * 10 = 10
1 * 11 = 11
1 * 12 = 12`

### Input 2
`2`

### Output 2
`2 * 1 = 2
2 * 2 = 4
2 * 3 = 6
2 * 4 = 8
2 * 5 = 10
2 * 6 = 12
2 * 7 = 14
2 * 8 = 16
2 * 9 = 18
2 * 10 = 20
2 * 11 = 22
2 * 12 = 24`

### Explanation
- For input `N = 1`, the multiplication table from `1` to `12` is printed.
- For input `N = 2`, each line shows `2 * i` where `i` is from `1` to `12`.
