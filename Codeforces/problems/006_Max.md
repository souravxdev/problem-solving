# Max
**Sheet #2 - Loops**

**Time Limit:** 1 second  
**Memory Limit:** 256 MB  

### Problem Statement
Given a number `N`, and then `N` integers, find and print the **maximum** number among them.

### Constraints
`1 ≤ N ≤ 10^3`  
`0 ≤ Xi ≤ 10^9`

### Input Format
- The first line contains the integer `N`.  
- The second line contains `N` integers `Xi`.

### Output Format
- Print the **maximum** number among the given inputs.

### Examples
### Input 1
`5
1 8 5 7 5`

### Output 1
`8`

### Explanation
Among the numbers `1, 8, 5, 7, 5`, the maximum value is **8**, so the output is `8`.

### Solution
The solution is implemented in **C language**, by reading all numbers and finding the maximum using a simple loop.
