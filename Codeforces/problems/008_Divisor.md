# Divisors
**Sheet #2 - Loops**

**Time Limit:** 1 second  
**Memory Limit:** 256 MB  

### Problem Statement
Given a number `N`, print all **positive divisors** of `N` in **ascending order**.  
A divisor of a number is a value that divides the number exactly with **no remainder**.

### Constraints
`1 ≤ N ≤ 10^4`

### Input Format
- Only one line containing an integer `N`.

### Output Format
- Print all **positive divisors** of `N`, each on a new line.

### Examples
### Input 1
`6`

### Output 1
`1
2
3
6`

### Input 2
`7`

### Output 2
`1
7`

### Input 3
`4`

### Output 3
`1
2
4`

### Explanation
A divisor is a number that divides another number **exactly** (no fractions or remainder).  
Examples:  
- `3` is a divisor of `12` because `12 ÷ 3 = 4` exactly.  
- `4` is a divisor of `12` because `12 ÷ 4 = 3` exactly.  
- `5` is **not** a divisor of `12` because `12 ÷ 5` leaves remainder `2`.

### Solution
The solution is implemented in **C language**, using a loop to check all numbers from `1` to `N`.
