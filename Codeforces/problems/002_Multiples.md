# Multiples
**Sheet #1 - Data Type - Conditions**

**Time Limit:** 1 second  
**Memory Limit:** 256 MB  

### Problem Statement
Given two numbers `A` and `B`.  
Print **"Multiples"** if `A` is a multiple of `B` **or** `B` is a multiple of `A`.  
Otherwise, print **"No Multiples"**.

### Constraints
`1 ≤ A, B ≤ 10^6`

### Input Format
- Only one line containing two integers `A` and `B`.

### Output Format
- Print **"Multiples"** if one number is a multiple of the other.  
- Otherwise print **"No Multiples"**.

### Examples
### Input 1
`9 3`

### Output 1
`Multiples`

### Input 2
`6 24`

### Output 2
`Multiples`

### Input 3
`12 5`

### Output 3
`No Multiples`

### Explanation
- In the first example, `9` is divisible by `3`, so the output is **Multiples**.  
- In the second example, `24` is divisible by `6`, so the output is **Multiples**.  
- In the third example, neither `12` is divisible by `5`, nor `5` is divisible by `12`, so the output is **No Multiples**.
